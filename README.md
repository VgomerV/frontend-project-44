ОПИСАНИЕ:
«Игры разума» — набор из пяти консольных игр, построенных по принципу популярных мобильных приложений для прокачки мозга. Каждая игра задает вопросы, на которые нужно дать правильные ответы. После трех правильных ответов считается, что игра пройдена. Неправильные ответы завершают игру и предлагают пройти ее заново. Игры:

Калькулятор. Арифметические выражения, которые необходимо вычислить.
Прогрессия. Поиск пропущенных чисел в последовательности чисел.
Определение четного числа.
Определение наибольшего общего делителя.
Определение простого числа.

УСТАНОВКА:
npm link

ЗАПУСК:
Для запуска игры "Калькулятор" выполните brain-calc
Для запуска игры "Арифметическая прогрессия" выполните brain-progression
Для запуска игры "Проверка на чётность" выполните brain-even
Для запуска игры "Определение наибольшего общего делителя" выполните brain-gcd
Для запуска игры "Простое ли число?" выполните brain-prime

ПРЕЗЕНТАЦИЯ ИГР:
Игра "Проверка на чётность". Вам показывается случайное число. Нужно ответить yes, если число чётное, или no — если нечётное. 
[![asciicast](https://asciinema.org/a/dHTc7w3r99fwjwkECdJJ35HKV.svg)](https://asciinema.org/a/dHTc7w3r99fwjwkECdJJ35HKV)

Игра "Калькулятор". Вам показывается математическое выражение, например 35 + 16, которое нужно вычислить и записать правильный ответ. 
[![asciicast](https://asciinema.org/a/WwRjlvOuKCvnMRBzBb22xiDGf.svg)](https://asciinema.org/a/WwRjlvOuKCvnMRBzBb22xiDGf)

Игра "НОД". Вам показывается два случайных числа, например, 25 50. Необходимо вычислить и ввести наибольший общий делитель этих чисел. 
[![asciicast](https://asciinema.org/a/S5vLk6MaxPpKJm93GBfzvrOCa.svg)](https://asciinema.org/a/S5vLk6MaxPpKJm93GBfzvrOCa)

Игра "Арифметическая прогрессия". Вам показывается прогрессия, необходимо определить пропущенное значение.
[![asciicast](https://asciinema.org/a/TkK6U8zcoqdk0YoRtqFq2bTed.svg)](https://asciinema.org/a/TkK6U8zcoqdk0YoRtqFq2bTed)

Игра "Простое ли число?". Вам показывается случайное число. Необходимо определить, простое оно или нет. Да - нужно ответить yes, нет - нужно ответить no.
[![asciicast](https://asciinema.org/a/E0P3cPEBKsyHK8AudMqgiy57q.svg)](https://asciinema.org/a/E0P3cPEBKsyHK8AudMqgiy57q)
